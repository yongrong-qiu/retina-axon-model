## Overview
The `Looming` directory contains analyses related to Figure 7. Looming stimuli are visual patterns that simulate an object approaching the observer, often used in neuroscience to study visual processing and threat detection mechanisms.

## Directory Contents
- **get_model_predictions_to_looming.ipynb**: code to get digital-twin model predictions to looming stimuli
- **looming_parametric_analysis.ipynb**: parametric analysis of looming responses
- **stimuli**: contains looming stimuli raw files
- **response_type_labels.csv**: response_type_labels from analysis in Figure 2
- **model_predicted_looming_responses.pkl**: has responses of all axonal boutons to looming stimuli
